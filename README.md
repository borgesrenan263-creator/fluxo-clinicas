# Fluxo Clínicas

Sistema Ruby para automação ética de reativação de contatos, atendimento e relatórios para clínicas e empresas locais.

## Recursos iniciais

- Cadastro de contatos
- Funil por status
- Templates de campanha
- Dashboard
- Relatório em PDF
- Banco SQLite local

## Rodar

bundle install
ruby db/setup.rb
ruby app.rb

## Acessar

http://localhost:4567

ou

http://127.0.0.1:4567

## Aviso

Use apenas bases de contatos legítimas da própria clínica/empresa.
Respeite pedidos de pausa, remoção ou "não perturbar".
